Podemos observar que tenemos un total de 5 entidades, usuario, coche, circuito, valoracion y alquiler. Alquileres es un array de alquileres y pertenece al usuario. Cada usuario tiene un array de con sus alquileres. Y cada alquiler depende de un coche y un circuito. Por ultimo cada valoracion pertenece a un cliente.
